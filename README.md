
# HomeAssistant

Integrate HomeAssistant into your Flexbar

## Installation


### **Prerequisites**

- Node.js 18 or later  
- FlexDesigner v1.0.0 or later  
- A Flexbar device 
- Install FlexCLI  
  ```
  npm install -g @eniac/flexcli
  ```

### Clone & Setup

```
git clone https://github.com/Sese-Schneider/flexbar-home-assistant.git
cd HomeAssistant
npm install
```

## Debug

```
npm run dev
```

## Build & Pack

```
yarn build
yarn plugin:pack
```
